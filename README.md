# 🏥 JEEVANsetu Healthcare System

JEEVANsetu is a full-stack healthcare simulation platform designed to model real-world hospital operations across multiple cities. It provides a dynamic dashboard for managing patient flow, hospital resources, and system load.

---

## 🚀 Features

* 🏙️ Multi-city hospital simulation (Delhi, Mumbai, Bangalore)
* 🛏️ Bed management and occupancy tracking
* 👨‍⚕️ Doctor workload and availability monitoring
* 🧾 OPD queue management with priority handling
* 📦 Inventory tracking (PPE, medicines, equipment)
* 🧠 Smart admission decision system
* 📊 Interactive dashboard for system insights

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Backend:** Node.js, Express
* **Database:** MongoDB Atlas

---


### 1. Install dependencies

```bash
cd backend && npm install
cd ../frontend && npm install
```

### 2. Configure environment variables

Create a `.env` file in backend:

```env
MONGODB_URI=your_mongodb_uri
PORT=5000
```

### 3. Run the project

```bash
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev
```
